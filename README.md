# AccessGuard

Enterprise Access Governance & Recertification Platform built with Spring Boot and PostgreSQL.

## Tech Stack

- Java 23
- Spring Boot 3.2.5
- PostgreSQL 16
- Spring Data JPA
- Flyway
- Docker
- Spring Security
- JWT (In Progress)

## Current Features

- User Persistence
- BCrypt Password Hashing
- REST APIs
- PostgreSQL Integration
- Flyway Database Migrations

## Architecture

Controller
→ Service
→ Repository
→ PostgreSQL

## Roadmap

- Registration API
- JWT Authentication
- Role-Based Access Control (RBAC)
- Access Requests
- Audit Events
- Access Governance Workflows
